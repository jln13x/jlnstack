# Built-in Plugins

Store comes with several built-in plugins that extend functionality.

## Installation

Plugins are exported from `@jlnstack/store`:

```ts
import { history, immer, logger, reset } from "@jlnstack/store"
```

## reset

Adds a `reset()` method to restore the store to its initial state.

```ts
import { createStore, reset } from "@jlnstack/store"

const { extensions, setState, getState } = createStore({
  state: { count: 0 },
  plugins: [reset()],
})

setState({ count: 10 })
extensions.reset.reset()
getState() // { count: 0 }
```

## history

Adds undo/redo functionality to your store.

```ts
import { createStore, history } from "@jlnstack/store"

const { extensions, setState, getState } = createStore({
  state: { count: 0 },
  plugins: [history()],
})

setState({ count: 1 })
setState({ count: 2 })

extensions.history.undo()
getState() // { count: 1 }

extensions.history.redo()
getState() // { count: 2 }
```

### Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `limit` | `number` | `100` | Maximum number of history states |

### Methods

| Method | Description |
|--------|-------------|
| `undo()` | Reverts to previous state |
| `redo()` | Re-applies undone state |
| `clear()` | Clears all history |
| `canUndo()` | Returns `true` if undo is available |
| `canRedo()` | Returns `true` if redo is available |
| `pastStates()` | Returns array of past states |
| `futureStates()` | Returns array of future states |

## immer

Enables mutative state updates using [Immer](https://immerjs.github.io/immer/).

```ts
import { createStore, immer } from "@jlnstack/store"

const { setState, actions } = createStore({
  state: { items: [] as string[] },
  actions: (set) => ({
    addItem: (item: string) =>
      set((s) => {
        s.items.push(item) // mutative update
      }),
  }),
  plugins: [immer()],
})

setState((s) => {
  s.items.push("hello")
})
```

## logger

Logs state changes to the console with action names.

```ts
import { createStore, logger } from "@jlnstack/store"

const { actions } = createStore({
  state: { count: 0 },
  actions: (set) => ({
    increment: () => set((s) => ({ count: s.count + 1 })),
  }),
  plugins: [logger({ name: "CounterStore" })],
})

actions.increment()
// Console: CounterStore increment { count: 1 }
```

### Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `name` | `string` | required | Store name for log prefix |
| `enabled` | `boolean` | `true` | Enable/disable logging |

## Combining Plugins

Use multiple plugins together:

```ts
const { extensions, actions } = createStore({
  state: { count: 0 },
  actions: (set) => ({
    increment: () => set((s) => ({ count: s.count + 1 })),
  }),
  plugins: [
    immer(),
    history(),
    reset(),
    logger({ name: "MyStore" }),
  ],
})

actions.increment()
extensions.history.undo()
extensions.reset.reset()
```
